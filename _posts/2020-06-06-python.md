## List
> extend()方法：用于在列表末尾一次性追加另一个序列中的多个值（用新列表来扩展原来的列表）  

```ruby
aList = [2020, 'xlp', 'create']
bList = ['zara', 'and', 'fresh']
aList.extend(bList)

print('Extend list : ', aList)
# => Extend list :  [2020, 'xlp', 'create', 'zara', 'and', 'fresh']
```
