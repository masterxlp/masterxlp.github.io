## Vi
### 01. 统计字符串在文本中出现的次数
在Linux系统上工作时，常常需要对日志文件进行分析、统计，这时就会涉及到想要统计某个字符串在文本中到底出现了多少次的需求。
在vi中可以是通过以下命令来完成对字符串的统计：

```
// 在整个文本中统计字符串出现的次数
:%s/字符串//ng

// 在整个文本的第m行到第n行统计字符串出现的次数
// 注意：在vi中“<>”符号是需要转义才能使用
:m,ns/\<字符串\>//gn

// 当然也可以借助Linux命令来实现该功能
// grep 统计字符串出现的行 wc -l 统计出现的次数
cat filename | grep -i 字符串 | wc -l
```
